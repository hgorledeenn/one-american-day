# One American Day
An analysis of one American day, based on data from the American Time Use Survey



*I found the top 5 activities at every interval (individual activities), then assigned them to larger categories (eg. leisure instead of computer use/watching tv/etc). Then, I re-plot with grouping by those larger categories. So all of the colorful boxes represent the people engaged in the top 5 most common activities at that time, but there might not be 5 different larger categories listed.*
